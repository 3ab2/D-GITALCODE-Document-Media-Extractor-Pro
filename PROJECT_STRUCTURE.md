## Data Flow

### Extraction Process
```
User Input (File/Folder)
    ↓
GUI validates selection
    ↓
ImageExtractor.extract_from_file/folder()
    ↓
Process each document:
    - Read document structure
    - Extract images from body
    - Extract from headers/footers
    - Calculate SHA256 hash
    - Check for duplicates
    - Detect image format
    - Save to appropriate folder
    ↓
Collect statistics
    ↓
ReportGenerator.generate_all_reports()
    ↓
Display results to user
```

## Key Algorithms

### Duplicate Detection
```
Image Data → SHA256 Hash → Known Hashes Set → Found? → Save to Duplicates or Format Folder
```

### Format Detection
```
Image Bytes → PIL Open → Format Property → Map to Folder → Return Extension + Folder
```

### File Processing
```
Folder → Find *.docx + *.doc → Validate → Extract → Aggregate Results → Generate Reports
```

## Configuration Points

To customize the application, edit:

**gui.py**
- Window size: `DEFAULT_WINDOW_WIDTH`, `DEFAULT_WINDOW_HEIGHT`
- Output location: Search for `desktop / "Extracted_Images_D-GITALCODE"`

**config.py**
- Colors, timeouts, performance settings

**utils.py**
- Languages, format support

## Extension Points

To extend functionality:

1. **Add Language**: Update `LANGUAGES` dict in `utils.py`
2. **Add Report Format**: Create method in `ReportGenerator` class
3. **Add File Type**: Update `SUPPORTED_DOC_FORMATS` in `config.py`
4. **Add Image Format**: Update format detection in `ImageDetector`
